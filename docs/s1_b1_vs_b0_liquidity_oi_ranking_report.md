# S1 B1 回测归因报告：流动性/持仓量排序相对 B0 的真实增量

## 1. 期权专家审议摘要

本次 B1 全样本回测已经完成，区间为 2022-01-04 至 2026-03-31。B1 相对 B0 的唯一策略变化是开仓优先级：在 B0 的全品种、次月、低 Delta、双侧卖权、50% 保证金、2.5x 权利金止损框架下，把品种和合约的优先级改为成交量与持仓量更高者优先。换句话说，B1 不是 IV/RV、趋势、降波或权利金质量模型，它只是一个更接近实盘可交易性的基准。

结论非常明确：B1 在净值、回撤、Sharpe、Calmar 上显著优于 B0。B1 累计收益 15.20%，CAGR 2.98%，最大回撤 -2.49%，Sharpe 1.53，Calmar 1.19；B0 同期累计收益 8.05%，CAGR 1.62%，最大回撤 -3.66%，Sharpe 0.64，Calmar 0.44。B1 相对 B0 多赚 7.15pct，同时最大回撤改善 1.16pct，最差单日从 B0 的 -2.66% 改善到 B1 的 -1.17%。

但从卖权质量看，B1 还不能被判定为最终优秀策略。B1 的 theta 累计 +987.17 万，高于 B0 的 +836.97 万；但 vega 累计 -588.52 万，比 B0 多亏约 -98.93 万；gamma 累计 -671.10 万，比 B0 多亏约 -71.71 万。B1 净值更好，主要来自更多、更稳定的 theta 流、较好的 residual、可交易性改善和尾部减轻，但 vega/gamma 对权利金的吞噬仍然很重。

核心判断是：B1 是一个明显更好的“可交易性基准”，但不是“卖波质量优化”的终点。它证明了合约流动性和持仓量排序很重要，也说明 B0 中一部分亏损可能来自低流动性、脏价格、边缘合约或开仓顺序问题；但 B1 没有解决 S1 的主矛盾：收来的 theta 仍然被 short vega 和 short gamma 大量吞噬。下一版 B2 应以“权利金质量排序”为主线，而不是继续单纯放大仓位。

## 2. 策略定义与对照口径

B0 配置文件：

```text
config_s1_baseline_b0_all_products_stop25.json
```

B1 配置文件：

```text
config_s1_baseline_b1_liquidity_oi_rank_stop25.json
```

B1 相对 B0 保持不变的规则包括：

- 全品种扫描。
- 只卖次月。
- 只卖虚值期权。
- 卖腿 abs(delta) <= 0.10。
- Put 和 Call 都尝试开仓，不主动趋势择边。
- 目标保证金使用率约 50%。
- 每个品种每侧最多 5 个合约。
- 使用真实手续费表和保证金率表。
- 不止盈。
- 持有到期，或权利金上涨到开仓权利金 2.5x 触发止损。
- 保留盘中止损跳变确认逻辑。

B1 唯一变化：

```text
B1 把 B0 的品种/合约优先级改为成交量与持仓量排序。
它不是 alpha 评分，也不是权利金质量评分。
```

## 4. Greek 与权利金质量归因

| 归因项 | B1 | B0 | B1 - B0 |
|---|---:|---:|---:|
| S1 PnL | +163.29 万 | +88.42 万 | +74.87 万 |
| Delta PnL | +17.01 万 | +28.57 万 | -11.56 万 |
| Gamma PnL | -671.10 万 | -599.39 万 | -71.71 万 |
| Theta PnL | +987.17 万 | +836.97 万 | +150.20 万 |

B1 的收益改善来自更强 theta 与 residual，而不是来自更好的 vega。B1 的 theta 增量 +150.20 万，足以覆盖 vega/gamma 额外损耗的一部分；residual 增量 +106.88 万也贡献很大。但这并不等于卖波质量改善，因为 vega 和 gamma 的绝对亏损都扩大了。

权利金质量指标更直接：

| 指标 | B1 |
|---|---:|
| 总开仓毛权利金 | 784.10 万 |
| 总开仓净权利金 | 776.40 万 |
| 日均开仓毛权利金 / NAV | 0.0602% |
| 开仓日均开仓毛权利金 / NAV | 0.0942% |
| Put 权利金占比 | 56.23% |
| vega loss / gross premium | 75.06% |
| gamma loss / gross premium | 85.59% |
| theta / gross premium | 125.90% |
| S1 PnL / gross premium | 20.83% |
| 已闭合权利金留存率 | 18.84% |

这组数字很关键：B1 收了 784 万毛权利金，但 vega 吞掉 589 万，gamma 吞掉 671 万。虽然 theta 比毛权利金更高，说明持有期间确实持续产生时间价值收入，但 vega/gamma 消耗过大，最终 S1 PnL 只相当于毛权利金的 20.83%。这意味着下一版核心不是“多卖”，而是“收来的保费要留得住”。

| 2023 | 4.83% |

年度层面看，B1 每年均为正，这是一个重要改善。它不像某些过拟合版本只靠单一年份支撑。但是收益分布仍偏“低波动、低年化”：稳定性比 B0 好，收益强度仍不足以接近乐得画像。

关键亏损日集中在 2022 年 6-7 月、2025 年 4 月、2026 年 3 月。最差单日为 2025-04-07，单日 -1.17%；其次是 2026-03-06 和 2026-03-09。这些日期大概率对应宏观事件、商品波动、关税/政策冲击或系统性升波。B1 的好处是把单日左尾压住了，但不是消灭了尾部。

## 6. 平仓路径与 P/C 结构

B1 的订单动作：

| 动作 | 笔数 | 手数 | PnL |
|---|---:|---:|---:|
| open_sell | 14220 | 70811 | 0 |
| expiry | 9066 | 44134 | +488.13 万 |
| sl_s1 | 4528 | 24096 | -345.46 万 |

持有到期贡献了主要正收益，止损吞掉了相当大一部分利润。这很符合卖权策略的典型画像：大量小赚，到期归零贡献正收益，少数止损吃掉多个到期周期的收益。B1 相比 B0 尾部更小，但止损规模仍然不低。

P/C funnel：

| 侧别 | 开仓毛权利金 | 已闭合权利金留存 | 留存率 | 止损订单占比 |
|---|---:|---:|---:|---:|
| Put | 440.93 万 | 83.81 万 | 19.88% | 34.20% |
| Call | 343.17 万 | 58.86 万 | 17.53% | 32.04% |

B1 并没有极端单侧化。Put 权利金占 56.23%，略高于 Call，但留存率也略好。与我们之前担心的“长期更多卖 Put 就等于看多所有品种”相比，B1 还算温和。不过 Put/Call 的留存率都不高，说明问题不是某一侧完全坏，而是整体保费质量仍有提升空间。

## 7. 品种与尾部贡献

开仓手数最多的品种包括 C、M、TA、RB、SR、MA、V、CF、RM、SA。B1 的流动性排序让主流商品更自然地进入组合，这符合实盘可交易性。但从乐得画像看，我们仍需检查 ETF/股指期权为什么占比不足，以及是否由于成交量/OI、Delta、权利金下限或次月合约口径导致金融期权自然靠后。

止损亏损较大的品种包括 M、CF、TA、RM、I、CU、RB、SR、AG、SA。尾部日实现亏损中，M-C、RM-C、CF-P、CU-P、M-P 等贡献靠前。尤其 CU-P、AG-P 在尾部日的 `pnl_per_open_premium` 极端偏差，说明少数金属品种/方向在尾部日可能有“权利金很薄、冲击赔付很大”的问题。这正是 B2 权利金质量排序要解决的典型场景。

### 8.4 PnL 归因

![PnL 归因](../output/analysis_s1_b1_liq_oi_rank_stop25_allprod_2022_latest/04_pnl_attribution.png)

- 怎么看：看 theta 是否覆盖 gamma/vega，residual 是否过大，delta 是否贡献了方向收益。

## 10. 与目标和乐得画像的差距

我们的目标是年化 6%、最大回撤小于 2%、并逐步接近乐得的卖权画像。B1 当前 CAGR 2.98%，最大回撤 -2.49%，没有达到目标。

与乐得画像相比，B1 的差距主要有四点：

- 收益强度不足：保证金接近 50%，但年化只有约 3%，说明单位保证金收益仍偏薄。
- vega 没有赚到：B1 的 vega 贡献显著为负，说明当前仍不是“降波收益捕捉”主导。
- 权利金留存率不足：已闭合权利金留存率 18.84%，保费赔付率仍高。
- 金融期权占比不足：B1 仍主要是商品活跃品种，和很多管理人大仓位配置股指/ETF 的画像不同。

## 10.1 金融期权、股指期权与 ETF 期权占比专项对比

用户特别关注“很多管理人会把股指/ETF 期权作为大仓位来源，为什么我们这里很少”。这件事必须单独拆出来，因为它会直接影响 S1 是否像乐得，也影响后续 B2 是否应该在金融期权上单独设预算。

本次统计先按实际成交/持仓结果分类：

```text
股指期权：IO、MO、HO
ETF 期权：产品名包含 ETF，或产品代码形态类似 510/159/588/515/512/513
商品期权：其他全部产品
```

实际结果很明确：

```text
B0 和 B1 都没有 ETF 期权开仓。
金融期权实际只有股指期权 IO、MO、HO。
```

也就是说，当前问题不是“ETF/股指占比低”这么温和，而是：

```text
ETF 期权占比为 0；
股指期权有进入，但占比仍然很低。
```

### 10.1.5 对策略设计的含义

这组结果对 B2 有直接影响：

第一，B2 不能只在所有品种里统一排序。若统一排序，商品因合约数量多、成交活跃、权利金形式多样，可能继续挤出金融期权。建议 B2 至少输出“商品/股指/ETF”三类独立分层结果。

第二，ETF 期权必须做数据可用性和候选漏斗诊断。当前实际为 0，不知道是没有进池，还是进池后被规则淘汰。下一步应在 candidate funnel 里加：

```text
ETF 产品数、ETF 次月可用合约数、ETF Delta 合格合约数、ETF 权利金大于手续费合约数、ETF 实际开仓数。
```

第三，金融期权可能需要独立预算，而不是完全和商品混在一起抢同一个边际预算。因为金融期权的风险性质、容量、成交质量、与商品相关性都不同。如果目标是更像乐得或主流管理人，金融期权不应只是自然排序的尾部结果。

第四，B2 的权利金质量排序应报告两个版本：

```text
全市场统一排序；
市场内部分层排序：商品内排序、股指内排序、ETF 内排序，再做组合预算分配。
```

如果二者差异很大，说明统一排序会产生结构性市场偏置。

B1 已经证明“可交易性排序”值得保留，但下一步必须进入“权利金质量排序”，即 B2。

## 11. 结论与下一步

B1 的策略结论：

```text
保留 B1 作为新的标准可交易性基准。
```
