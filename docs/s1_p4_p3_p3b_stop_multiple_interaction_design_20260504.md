# S1 P4 实验设计：P3/P3B × 止损倍数交互验证

日期：2026-05-04

## 1. 研究背景

上一轮产品池实验显示，P3B（乐得主流品种 + 期限偏好）是当前最接近目标画像的一条线：

| 版本 | 总收益 | 年化 | 最大回撤 | Sharpe | Calmar | 相对 P0 超额 |
|---|---:|---:|---:|---:|---:|---:|
| P0：B2C + 1.5X，全品种 | 22.98% | 4.39% | -2.24% | 2.95 | 1.96 | 0.00% |
| P3：乐得主流品种 | 26.57% | 5.01% | -2.67% | 2.49 | 1.87 | +3.59% |
| P3B：乐得主流品种 + 期限偏好 | 30.11% | 5.62% | -2.05% | 2.85 | 2.73 | +7.13% |

从收益公式看：

```text
S1 net return
= Premium Pool × Deployment Ratio × Retention Rate
- Tail / Stop Loss
- Cost / Slippage
```

P3B 的优势主要来自 `Premium Pool` 显著扩大，而不是 `Retention Rate` 明显提升。P3B 开仓毛权利金池约 1711 万，明显高于 P0 的 1181 万；但 P3B 权利金留存率约 18.08%，低于 P0 的 19.63%。这说明 P3B 是“保费池更厚但风险也更尖”的版本。

乐得口径中提到的止损规则是权利金上涨至原始权利金的 `2.5X`。我们之前在全品种/B2C 上接受了 `1.5X` 作为阶段性基准，但这个结论不一定能迁移到 P3/P3B，因为止损倍数和品种池、期限、流动性、权利金厚度、趋势尾部高度耦合。

因此，本轮 P4 实验不再单独讨论“1.5X 还是 2.5X 哪个更好”，而是验证：

> 在乐得主流产品池和期限偏好结构下，止损倍数的最优区域是否发生变化？

## 2. 实验核心假设

### H1：主流品种池会降低“假止损”比例，较宽止损可能提升留存率

P3/P3B 更集中于主流商品期权，理论上流动性和报价质量更好。若之前 1.5X 触发了过多噪声止损，那么在 P3/P3B 中放宽到 2.0X 或 2.5X 可能提高权利金留存率，并减少“止损后最终归零”的错杀。

### H2：期限偏好提高 Premium Pool，但也提高短 Gamma 暴露

P3B 的收益高于 P3，说明期限偏好确实提高了权利金池；但 P3B 的 Gamma 损耗也更重。若止损倍数从 1.5X 放宽到 2.5X，可能进一步提高留存率，也可能把短 Gamma 风险转化为更深回撤。

### H3：不止损不是目标线，而是风险边界诊断线

不止损版本不是候选实盘方案，而是用于回答：

- 1.5X/2.5X 是否真正减少尾部亏损；
- 止损是否只是把本来会归零的合约提前砍掉；
- 没有止损时最大回撤和极端月份会恶化到什么程度。

如果不止损收益明显更高但回撤不可接受，说明止损是必要的；如果不止损没有明显恶化，说明当前止损可能过紧。

## 3. 实验矩阵

### 3.1 产品池维度

| 产品池 | 定义 | 已有锚点 |
|---|---|---|
| P3 | 乐得主流商品期权名单 | `s1_b2c_stop15_ledet_mainstream_2022_latest` |
| P3B | P3 + 期限偏好：黄金双月、白银/铁矿近月、部分品种主力月，且满足 DTE 边界 | `s1_b2c_stop15_ledet_term_pref_2022_latest` |

P3/P3B 品种清单：

```text
CU, AL, ZN, SN, NI, AU, AG, EG, TA, MA, EB, SH, SA, FG, AO, SC, M, RM, B, P, RB, I, RU
```

### 3.2 止损倍数维度

| 档位 | 含义 | 作用 |
|---|---|---|
| 1.5X | 当前 P3/P3B 锚点 | 严格止损，控制短 Gamma 和异常波动 |
| 2.0X | 中间档 | 检查 1.5X 是否过紧 |
| 2.5X | 乐得经验档 | 核心验证档 |
| 3.0X | 宽止损 | 检查更宽止损是否继续改善留存或开始恶化尾部 |
| 不止损 | 边界诊断 | 判断止损制度本身的价值 |

### 3.3 需要新跑的 8 条线

1. P3 × 2.0X
2. P3 × 2.5X
3. P3 × 3.0X
4. P3 × 不止损
5. P3B × 2.0X
6. P3B × 2.5X
7. P3B × 3.0X
8. P3B × 不止损

`P3 × 1.5X` 和 `P3B × 1.5X` 复用已完成结果，不重复跑。

## 4. 控制变量

本轮只改变 `premium_stop_multiple`，其他保持不变：

- 样本：2022-01-01 至当前有数据最后一日。
- 资金：1000 万。
- 组合保证金上限：50%。
- 合约选择：B2C 质量预算框架。
- Delta：卖方腿小于 0.1。
- 成交与费用：沿用当前交易所/期货公司费用与滑点口径。
- 到期处理：到期日若未实值则价值归零；若实值按内在价值结算。
- 止损执行：继续使用当前分钟路径 + 异常价确认 + 日内最高价预筛。
- P3B 期限偏好：继续要求满足 DTE 边界，避免“只按月份/主力但期限过短或过长”。

## 5. 评价指标

### 5.1 一级指标：能否接近策略目标

- 年化收益是否接近或超过 6%。
- 最大回撤是否不超过 2%-2.3%。
- Sharpe、Calmar 是否不明显低于 P0。
- NAV 超额是否持续，而不是只靠某一段行情跳升。

### 5.2 收益公式指标

| 变量 | 观察指标 | 解释 |
---|---|---|
| Premium Pool | 开仓毛/净权利金、日均开仓权利金/NAV | 止损改变是否影响部署率和再开仓机会 |
| Deployment Ratio | 平均保证金、活跃品种、活跃合约、持仓手数 | 宽止损是否占用资金导致新仓减少 |
| Retention Rate | 权利金留存率、到期收益/开仓权利金 | 核心判断止损是否过紧 |
| Tail / Stop Loss | 止损次数、止损亏损、最大回撤、最差日/月 | 判断宽止损是否把显性止损变成更深浮亏 |
| Cost / Slippage | 手续费、止损滑点、费用/权利金 | 宽止损是否降低交易成本 |

### 5.3 Greek 与卖波质量指标

- Theta / gross premium。
- Vega loss / gross premium。
- Gamma loss / gross premium。
- Delta PnL 占比，判断是否变成方向收益。
- Residual 占比，判断是否存在口径或非线性遗漏。

### 5.4 止损后命运指标

本轮报告必须新增：

- 止损后最终归零比例。
- 止损后继续不利移动比例。
- 止损后同合约/同品种反向修复幅度。
- 2.5X 相比 1.5X 少触发的止损，最终是“应该忍”还是“应该砍”。

这组指标是判断乐得 2.5X 是否合理的关键。

## 6. 判断标准

### 6.1 可作为主线升级

满足以下条件之一：

- P3B × 2.5X 年化更接近 6%，最大回撤仍在 2%-2.3% 内，且 Calmar 不低于 P3B × 1.5X。
- P3B × 2.0X 或 2.5X 明显提高 Retention Rate，同时没有显著放大最差日和止损簇集。
- P3 × 2.5X 在更低回撤下接近 P3B 收益，说明期限偏好可以适当降强度。

### 6.2 只能作为诊断线

- 收益更高但最大回撤明显超过 2.5%。
- Vega/Gamma 损耗显著扩大，收益主要依赖 Delta 或 Residual。
- 不止损收益最高但尾部不可接受。

### 6.3 应该否决

- 宽止损没有提高留存率，反而增加止损亏损和回撤。
- 2.5X 与 3.0X 显著劣于 1.5X，说明乐得口径不能直接迁移到我们的执行和合约筛选系统。
- P3B 的优势只在少数品种/月份出现，且尾部相关性高度集中。

## 7. 预期解读方式

本轮最终不应只输出“哪个倍数最好”，而应输出二维结论：

| 结论维度 | 需要回答 |
---|---|
| 产品池 | P3 和 P3B 哪个更适合做主线 |
| 止损倍数 | 1.5X、2.0X、2.5X、3.0X 哪个更符合收益/回撤目标 |
| 交互关系 | P3B 是否需要比 P3 更严格或更宽的止损 |
| 卖波质量 | 提升来自 theta/vega，还是来自方向/残差 |
| 风险边界 | 不止损版本的最大尾部代价是什么 |

## 8. 预设研究判断

我的先验判断是：

- P3B × 2.0X 或 P3B × 2.5X 最值得期待。
- P3B × 3.0X 可能收益更高，但回撤和 gamma 损耗大概率会变差。
- P3 × 2.5X 可能更稳，但收益可能仍低于 P3B。
- 不止损大概率是诊断线，不适合作为候选主线。

如果 P3B × 2.5X 能把年化推到 6% 附近，同时最大回撤不超过 2.3%，它就可以成为下一版 S1 主线候选；之后再叠加组合层尾部相关性、板块预算和到期集中控制。
